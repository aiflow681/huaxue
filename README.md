户外滑雪 - 游戏操作说明
================================

【游戏简介】
这是一个基于HTML5 Canvas技术开发的轻量级滑雪游戏。玩家需要操控滑雪者躲避沿途的树木和岩石，尽可能滑行更远的距离以获取高分。本版本无需任何插件，支持PC和移动端直接运行。

【如何开始】
1. 找到游戏文件 "index.html"。
2. 双击文件，使用任意现代浏览器（如Chrome, Edge, Safari, Firefox）打开即可。
3. 游戏加载后会自动进入准备状态。

【操作指南】

[电脑端 (PC)]
- 向左移动：按下键盘 "←" (左箭头) 或 "A" 键。
- 向右移动：按下键盘 "→" (右箭头) 或 "D" 键。
- 重新开始：游戏结束后，点击屏幕上的“重新开始”按钮，或直接按 "Enter" (回车) 键。

[移动端 (手机/平板)]
- 向左移动：点击或长按屏幕左下方的 "⬅️" 按钮。
- 向右移动：点击或长按屏幕右下方的 "➡️" 按钮。
- 重新开始：游戏结束后，点击屏幕中央弹出的“重新开始”按钮。

【游戏规则】
1. 躲避障碍：滑雪过程中会出现大量的松树和岩石，撞上任意障碍物游戏即结束。
2. 获取分数：
   - 分数随着滑行距离自动增加。
   - 滑行速度越快，得分越快。
3. 难度递增：
   - 随着游戏进行，滑雪者的速度会逐渐加快。
   - 障碍物的生成密度也会越来越高，极具挑战性。

【高分技巧】
- 保持专注，提前预判障碍物的位置。
- 在高速度下，尽量保持在屏幕中央区域活动，以便有足够的反应空间向左或向右闪避。
- 不要大幅度频繁左右横跳，微调位置往往更安全。
