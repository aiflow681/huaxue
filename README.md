【游戏名称】
户外滑雪

【游戏介绍】
这是一款H5原生开发的休闲滑雪类小游戏。游戏中，你扮演一名滑雪者，在雪地上快速滑行。你需要左右移动躲避前方的树木和石头障碍，滑得越远得分越高！随着游戏进行，速度会越来越快，挑战你的反应能力。

【玩法说明】
游戏中，你的滑雪者会自动向前滑行，雪地上会不断出现树木和石头障碍物。

障碍：
1. 树木和石头会从屏幕下方不断出现，向上移动。
2. 如果撞到任何障碍物，游戏就会结束。
3. 随着得分增加，滑行速度会越来越快，障碍物出现频率也会提高。

操作：
点击屏幕左侧按钮（或按左方向键）- 向左移动
点击屏幕右侧按钮（或按右方向键）- 向右移动
按回车键 - 游戏结束后重新开始

目标：
躲避所有障碍物，尽可能滑得更远，获得更高的分数！

Tips：游戏会自动记录你的得分。
速度越快，得分增长越快。提前预判障碍物位置，保持冷静，挑战最高分！

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用了Canvas 2D渲染技术，实现了流畅的动画效果和碰撞检测，纯原生实现无需任何框架。
